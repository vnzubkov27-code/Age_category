# Модель предсказания возрастных категорий пользователей

## Описание проекта
Модель многоклассовой классификации для определения возрастной категории пользователей на основе цифрового следа. Модель классифицирует пользователей по 5 возрастным группам для таргетированной рекламы.

## Бизнес-задача
Разработка модели для рекламной сети «Йети» для:
- Персонализации рекламы для разных возрастных групп
- Соблюдения возрастных ограничений
- Увеличения конверсии рекламных кампаний
- Повышения удовлетворенности пользователей релевантностью рекламы

## Возрастные категории
- 0: Младше 18 лет
- 1: 18-25 лет
- 2: 26-40 лет
- 3: 41-55 лет
- 4: 56+ лет

## Источники данных
- Логи посещений сайтов (категории сайтов, время сессий)
- Активность взаимодействия с рекламой
- Паттерны глубины просмотра
- Информация об основном устройстве
- Данные об использовании облачных технологий

## Качество модели
- **F1-score (macro)**: 0.8314
- **Accuracy**: 0.8568
- **Precision (macro)**: 0.8277
- **Recall (macro)**: 0.8390

## Детали модели
- **Алгоритм**: SVM с RBF ядром
- **Лучшие параметры**: C=1.0, gamma=0.1, class_weight='balanced'
- **Валидация**: 5-кратная стратифицированная кросс-валидация
